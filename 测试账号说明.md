## 注意事项

1. **密码加密：** 密码使用BCrypt加密，哈希值为：`$2a$10$N9qo8uLOickgx2ZMRZoMyeIjZAgcfl7p92ldGxad68LJZdL17lhWy`
   - 对应明文密码：`123456`
   - 如果需要修改密码，可以使用在线BCrypt生成器：https://bcrypt-generator.com/

2. **用户ID格式：** `U2026012800001`
   - U：用户前缀
   - 20260128：日期（2026年1月28日）
   - 00001：序号

3. **钱包ID格式：** `W2026012800001`
   - W：钱包前缀
   - 与用户ID对应

4. **角色ID：** 使用默认角色 `1` (ROLE_USER - 普通用户)

## 如果账号已存在

如果执行时提示用户名或邮箱已存在，可以：

1. **删除旧账号：**
```sql
-- 删除钱包（先删钱包，因为有外键约束）
DELETE FROM `user_wallets` WHERE `user_id` = 'U2026012800001';
-- 删除用户
DELETE FROM `users` WHERE `id` = 'U2026012800001';
```

2. **或者修改用户名和邮箱：**
```sql
-- 修改用户名和邮箱
UPDATE `users` 
SET `username` = 'testuser2', 
    `email` = 'testuser2@example.com'
WHERE `id` = 'U2026012800001';
```

## 生成其他测试账号

如果需要生成更多测试账号，可以修改SQL中的：
- `id`: 修改日期和序号
- `username`: 修改用户名
- `email`: 修改邮箱
- `password_hash`: 使用新的BCrypt哈希值
